# AuroraBeat 打包安装指南

本文档介绍如何将 AuroraBeat 音乐播放器打包成可以安装到电脑上的软件。

## 📋 前置条件

在打包之前，请确保你的电脑已安装以下软件：

1. **Node.js** - 版本 18.x 或更高（推荐 LTS 版本）
   - 下载地址: https://nodejs.org/
   - 安装后打开命令行输入 `node -v` 验证

2. **npm** - 随 Node.js 一起安装
   - 验证命令: `npm -v`

3. **Git**（可选，推荐）
   - 下载地址: https://git-scm.com/

## 💻 Windows 系统打包

### 方法一：使用打包脚本（推荐）

1. 将项目文件复制到你的 Windows 电脑上

2. 双击运行 `build-windows.bat` 文件

3. 按照提示选择打包方式：
   - **1. 安装包 (NSIS .exe)** - 生成 .exe 安装程序，用户双击即可安装
   - **2. 便携版 (Portable .exe)** - 生成单个 .exe 文件，无需安装直接运行
   - **3. 文件夹形式** - 生成文件夹，里面有 AuroraBeat.exe，直接双击运行
   - **4. 全部生成** - 生成以上所有版本

4. 等待打包完成，成品在 `release` 文件夹中

### 方法二：手动命令行打包

1. 打开命令提示符（CMD）或 PowerShell

2. 进入项目目录：
   ```cmd
   cd C:\path\to\aurorabeat
   ```

3. 设置国内镜像（加速下载）：
   ```cmd
   set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
   set ELECTRON_BUILDER_BINARIES_MIRROR=https://npmmirror.com/mirrors/electron-builder-binaries/
   ```

4. 安装依赖：
   ```cmd
   npm install --registry=https://registry.npmmirror.com
   ```

5. 开始打包：
   ```cmd
   npm run build:win
   ```

6. 打包完成后，安装包在 `release` 文件夹中

### Windows 打包产物说明

| 文件 | 说明 | 使用方式 |
|------|------|----------|
| `AuroraBeat Setup 1.0.0.exe` | NSIS 安装包 | 双击运行，按向导安装 |
| `AuroraBeat 1.0.0.exe` | 便携版 | 双击直接运行，无需安装 |
| `win-unpacked/` | 解压版文件夹 | 打开文件夹，双击 AuroraBeat.exe |

## 🍎 macOS 系统打包

### 方法一：使用打包脚本（推荐）

1. 将项目文件复制到你的 Mac 电脑上

2. 打开「终端」，进入项目目录：
   ```bash
   cd /path/to/aurorabeat
   ```

3. 给脚本添加执行权限：
   ```bash
   chmod +x build-mac.sh
   ```

4. 运行脚本：
   ```bash
   ./build-mac.sh
   ```

5. 按照提示选择打包方式

6. 打包完成后，成品在 `release` 文件夹中

### 方法二：手动命令行打包

1. 打开终端，进入项目目录

2. 设置国内镜像：
   ```bash
   export ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
   export ELECTRON_BUILDER_BINARIES_MIRROR=https://npmmirror.com/mirrors/electron-builder-binaries/
   ```

3. 安装依赖：
   ```bash
   npm install --registry=https://registry.npmmirror.com
   ```

4. 开始打包：
   ```bash
   npm run build:mac
   ```

### macOS 打包产物说明

| 文件 | 说明 | 使用方式 |
|------|------|----------|
| `AuroraBeat-1.0.0.dmg` | DMG 安装包 | 双击打开，拖拽到应用程序文件夹 |
| `mac/` | 解压版 | 打开文件夹，拖拽 AuroraBeat.app 到应用程序 |

## ⚡ 常见问题

### 1. 下载依赖很慢或失败？

**原因**：默认从国外服务器下载，速度慢

**解决方法**：使用国内镜像
```cmd
npm install --registry=https://registry.npmmirror.com
```

### 2. 打包时下载 Electron 很慢？

**原因**：Electron 二进制文件较大，从国外下载慢

**解决方法**：设置镜像后再打包
```cmd
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
set ELECTRON_BUILDER_BINARIES_MIRROR=https://npmmirror.com/mirrors/electron-builder-binaries/
npm run build:win
```

### 3. 提示 "electron failed to install correctly"？

**原因**：Electron 没有正确安装

**解决方法**：
1. 删除 `node_modules` 文件夹
2. 删除 `package-lock.json` 文件
3. 设置镜像后重新安装：
   ```cmd
   set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
   npm install --registry=https://registry.npmmirror.com
   ```

### 4. 打包时 winCodeSign 下载失败？

**原因**：Windows 代码签名工具下载失败

**解决方法**：
- 设置 electron-builder 二进制镜像（见上文）
- 或者手动下载 winCodeSign 放到缓存目录

### 5. 第一次打包特别慢？

**原因**：第一次需要下载 Electron 和各种工具

**解决方法**：耐心等待，第二次打包就会快很多，因为有缓存了

## 🎯 快速开始（最简单的方式）

如果你只是想在自己电脑上运行，不需要打包成安装包：

1. 安装 Node.js
2. 打开命令行，进入项目目录
3. 运行：
   ```cmd
   set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
   npm install --registry=https://registry.npmmirror.com
   npm run dev:electron
   ```
4. 软件就会启动了！


## 🔧 自定义配置

### 修改软件名称/版本

编辑 `package.json` 文件：
```json
{
  "name": "aurorabeat",
  "version": "1.0.0",
  "productName": "AuroraBeat"
}
```

### 修改应用图标

1. 准备你的图标文件
2. Windows: 准备 256x256 的 `.ico` 文件
3. macOS: 准备 `.icns` 文件
4. 放到 `public/` 目录，命名为 `icon.ico` 和 `icon.icns`

### 修改安装包配置

编辑 `package.json` 中的 `build` 部分，可以自定义：
- 安装向导界面
- 开始菜单名称
- 桌面快捷方式
- 文件关联
- 等等

## 📞 需要帮助？

如果遇到问题，请检查：
1. Node.js 版本是否正确（>= 18.x）
2. 网络连接是否正常
3. 磁盘空间是否足够（至少 2GB）
4. 是否有杀毒软件拦截
